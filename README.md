# SlideAkmalV1

Laman slide interaktif **Slide Akmal V1** yang meniru gaya deck Custom Gemini AUD: latar putih-biru yang bersih, aksen emas, progress dots, kawalan keyboard, fullscreen, dan layout slide terkunci kepada viewport.

## Laman Langsung

https://akmal4244.github.io/SlideAkmalV1/

## Kandungan Awal

- Slide pembuka untuk Slide Akmal V1
- Matlamat pembentangan
- Konteks, isu, dan keputusan yang perlu dibawa
- Aliran naratif slide
- Struktur kandungan
- Gaya visual yang dikekalkan daripada CustomGeminiAUD
- Ruang prompt atau nota pembentangan yang boleh disalin
- Pelan tindakan dan senarai semak
- Penutup

## Pembangunan Lokal

```powershell
npm test
npm start
```
